# jetbrains-deimos

JetBrains plugin for Deimos Code (IntelliJ, WebStorm, PyCharm, GoLand, etc.)

## Status

Scaffold — plugin implementation pending.

## Planned

- Tool window with embedded terminal running `deimos code`
- Right-click > "Ask Deimos" on any selection
- File > "Explain with Deimos"
- Shortcut: Ctrl+Shift+D / Cmd+Shift+D

## Development

Requires IntelliJ Platform Plugin SDK (Kotlin/Java).
See [intellij-platform-plugin-template](https://github.com/JetBrains/intellij-platform-plugin-template).
